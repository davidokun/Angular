# Angular Examples

This repo contains multiple projects using Angular 5 framework.

## Useful CLI commands

Install Angular CLI:
- `npm install -g @angular/cli`

Create new project:
- `ng new project-name` 

Run project:
- `ng serve`

Create new Component:
- `ng generate component component-name`
- `ng g c component-name`

Install bootstrap
- `ng install --save bootstrap`
